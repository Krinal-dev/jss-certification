# Icons

This project includes Font Awesome, the Internet's icon library and toolkit, used by millions of designers, developers, and content creators. For additional information on the Font Awesome library, visit <a href="https://fontawesome.com">fontawesome.com</a>.

Icons can be included in any React component by including the following code snippet:

<Source
  code={`
import { FontAwesomeIcon } from '@fortawesome/react-fontawesome';
import { faBars } from '@fortawesome/free-solid-svg-icons';

<FontAwesomeIcon icon={faBars} />
`} />
